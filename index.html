Projects

  • The WAN Project:

    Description:

    In this project, I successfully designed a Wide Area Network (WAN) that connected multiple local area networks (LANs) across different geographical locations. The project involved:

    • Research and Planning: Conducted extensive research on WAN technologies and protocols (e.g., MPLS, VPN, Frame Relay) to determine the most suitable options for the project requirements.
    • Network Design: Created detailed network diagrams using tools like Cisco Packet Tracer and Visio to visualize the network architecture, including routers, switches, and communication links.
    • Configuration: Configured networking devices to establish connectivity between sites, ensuring optimal data flow and security measures were in place.
    • Testing and Validation: Implemented testing procedures to validate network performance and reliability, using tools like Wireshark, ping, and traceroute to troubleshoot and optimize the network.
    • Documentation: Prepared comprehensive documentation outlining the network design, configuration steps, and testing results, which provided a valuable reference for future network implementations.

    Skills Developed:

    • Network design and architecture
    • Configuration and troubleshooting of networking devices
    • Understanding of WAN technologies and protocols
    • Documentation and communication skills
